AI-Driven Problem Solving Portfolio

5개 제품에서 반복 업무를 자동화하고, 비정형 데이터를 구조화해 실행 가능한 인사이트로 연결한 문제해결 사례입니다.

공통 문제와 공통 해법

Problem Pattern

5개 프로젝트 모두 비정형 정보가 흩어져 있어, 정리/분석/재사용이 어려운 문제를 다룹니다.

Solution Pattern

사용자 입력과 외부 데이터를 구조화하고, AI 분석 결과를 실제 액션 가능한 형태로 연결했습니다.

Impact Pattern

반복 작업을 줄이고 의사결정 속도를 높이며, 개인 지식/기록 자산의 누적 가능성을 강화했습니다.

AI 활용 원칙 (요약)

Collection → Normalize → Infer → Serve → Improve
  • 입력 데이터를 먼저 구조화하고, AI는 분석/추론 단계에 집중 적용
  • 모델 출력은 UI/저장/피드백 루프로 반드시 연결
  • 사용자 반복 작업이 줄어드는지로 효과를 검증

AI Problem-Solving Flow (Across 5 Cases)

1. Problem Framing 반복/누락/비효율 작업 정의
→
2. Data Collection 텍스트/기록/외부 입력 수집
→
3. Normalization 스키마화, 필드 매핑, 정형화
→
4. AI Inference 요약/추출/분석/평가 수행
→
5. Serving & Feedback UI 반영, 저장, 재분석 루프

Case Studies

Case 01. Blog Knowledge System

솔루션

Jekyll 기반의 데이터 엔지니어링 기술 블로그.

문제

학습/실무 지식이 산발적으로 남아 재사용이 어려운 문제를 해결했습니다.

AI 적용

포스트 초안 작성, 구조화, 문장 다듬기 등 콘텐츠 생산 보조에 활용했습니다.

모델
LLM | Draft Generation LLM | Summarization LLM | Style Refinement
결과

발행 가능한 초안과 구조화된 기술 아티클을 생성합니다.

핵심 기여
  • 기술 글 아카이빙 구조를 만들어 학습 자산을 누적 가능하게 구성
  • 주제별 글 흐름을 표준화해 작성/발행 리드타임 단축

Case 02. CareerWeb Automation

솔루션

채용공고 수집, Notion 정리, ATS 분석까지 연결한 작업 관리 웹앱 (React + FastAPI).

문제

채용공고를 수동으로 정리/비교/분석하는 반복 업무를 자동화했습니다.

AI 적용

LLM 기반 공고 파싱, ATS 분석, 정리본 생성 및 Notion 동기화에 사용했습니다.

모델
Gemini | Job Posting Parsing Gemini | ATS Fit Analysis Gemini | Report Generation
결과

구조화된 공고 데이터와 ATS 피드백 리포트를 제공합니다.

핵심 기여
  • 공고 수집 → 분석 → 저장까지의 수작업 흐름을 자동화 파이프라인으로 전환
  • Notion 연동으로 분석 결과의 누적/검색/재사용 가능성 강화

Case 05. RecipeBook Structuring Engine

솔루션

레시피 저장/검색/조리 모드를 제공하는 iOS 레시피 관리 앱.

문제

온라인 레시피의 비정형 텍스트를 정리해 내 레시피로 재사용하기 어렵던 문제를 해결했습니다.

AI 적용

Gemini 파서로 레시피 텍스트를 구조화하고, 블로그 콘텐츠 파싱 자동화에 사용했습니다.

모델
Gemini | Recipe Parsing Gemini | Data Structuring Gemini | Content Extraction
결과

재사용 가능한 구조화 레시피 데이터를 생성합니다.

핵심 기여
  • 비정형 레시피 텍스트를 구조 데이터로 변환해 검색성과 재사용성 개선
  • 조리 모드/상세 뷰를 분리해 실제 조리 상황에서의 사용성 강화
